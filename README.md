## Google Search Console Setup

1. **Verify Your Site**
   - Go to https://search.google.com/search-console
   - Add property: your-domain.com
   - Verify using DNS TXT record or HTML file method

2. **Submit Sitemap**
   - Navigate to Sitemaps in left sidebar
   - Submit: https://your-domain.com/sitemap.xml

3. **Set Up URL Inspection**
   - Use URL Inspection tool to check indexing status
   - Request indexing for new pages

4. **Monitor Performance**
   - Track clicks, impressions, CTR, and average position
   - Identify top-performing pages and keywords
   - Monitor Core Web Vitals in Experience section

5. **Set Up Email Alerts**
   - Configure alerts for critical issues
   - Monitor coverage errors and indexing problems
